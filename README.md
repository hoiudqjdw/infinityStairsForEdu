# infinityStairsForEdu
무한의 계단 클론 코딩

전 세계적으로 큰 인기를 끌었던 모바일 게임 **무한의 계단(Infinity Stairs)**의 게임을 클론 코딩 합니다. [플레이 영상](https://www.youtube.com/shorts/XGYXYYoZEc8) 참고해 웹에서 할 수 있는 게임을 제작합니다. GitHub Pages 활용해 퍼블리싱할 예정입니다. 모바일, PC모두 플레이 가능해야 합니다. 핵심루프 중심으로 프로토타입을 제작합니다.

[스크린샷](screenshot/) 폴더 참고해 제작합니다. 모든 리소스는 이미지 없이 도트로 제작합니다.

효과음/BGM 갖고올 프로젝트: https://github.com/choijoohee213/Infinite-Stairs.git

## 파일 구조

초등학생들이 쉽게 열어서 수정할 수 있도록 HTML/CSS/JS를 `index.html` 한 파일에 모두 합쳐뒀습니다.
(`<style>` 태그 안에 CSS, `<script>` 태그 안에 JS가 들어있습니다.) 사운드 파일만 `audio/` 폴더에 따로 있습니다.
